# Tables
Tables is an easy library for the quick use of mysql database.

- On development

## .env
Required...
``` php
DB_TYPE=
DB_HOST=
DB_USER=
DB_PASS=
DB_DTBS=
DB_PORT=
```

## Required folder with entities definitions
Example...
```
name[varchar(255)]:null
surname[varchar(255)]:null
what[varchar(255)]:null
```
